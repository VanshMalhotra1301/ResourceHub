Fractional Knapsack

Time: O(N log N) Space: O(1) Greedy Algorithm

In the Fractional Knapsack problem, you are given a set of items, each with a weight and a value. You must maximize the total value in a knapsack of capacity W. Unlike 0/1 Knapsack, you are allowed to break items to take a fraction. The Greedy Choice is to always pick the item with the highest Value/Weight Ratio.

Greedy Strategy
function fractionalKnapsack(items, W):
for each item in items:
item.ratio ← item.value / item.weight
sort(items) descending by ratio
totalValue ← 0
for each item in items:
if W == 0: break
if item.weight ≤ W: ← FULL ITEM
W ← W - item.weight
totalValue ← totalValue + item.value
else: ← FRACTIONAL ITEM
fraction ← W / item.weight
totalValue ← totalValue + (item.value * fraction)
W ← 0
return totalValue

// Fractional Knapsack - Greedy Approach struct Item { int value, weight; }; // Comparator to sort items by Value/Weight ratio descending bool cmp(struct Item a, struct Item b) { double r1 = (double)a.value / (double)a.weight; double r2 = (double)b.value / (double)b.weight; return r1 > r2; } double fractionalKnapsack(int W, struct Item arr[], int n) { sort(arr, arr + n, cmp); double finalValue = 0.0; for (int i = 0; i < n; i++) { if (arr[i].weight <= W) { W -= arr[i].weight; finalValue += arr[i].value; } else { finalValue += arr[i].value * ((double)W / (double)arr[i].weight); break; // Knapsack is full } } return finalValue; }
