Insertion Sort

Best: O(n) Avg / Worst: O(n²) Space: O(1) In-Place Sorting

Insertion Sort builds the final sorted array one item at a time. It behaves exactly like sorting playing cards in your hands — taking one element from the unsorted deck and inserting it into its correct position within the already sorted cards.

Pseudocode
function insertionSort(arr):
for i ← 1 to len(arr) - 1:
key ← arr[i]
j ← i - 1
while j ≥ 0 and arr[j] > key:
arr[j + 1] ← arr[j] ← Shift right
j ← j - 1
arr[j + 1] ← key ← Insert key
Implementation
// Insertion Sort — O(n²) void insertionSort(vector<int>& arr) { for (int i = 1; i < arr.size(); i++) { int key = arr[i]; int j = i - 1; // Move elements that are greater than key // to one position ahead of their current position while (j >= 0 && arr[j] > key) { arr[j + 1] = arr[j]; j = j - 1; } arr[j + 1] = key; } }

How Insertion Sort Works
1
Initialization: Assume the first element is already sorted. The sorted partition is on the left.
2
Select the next unsorted element as the key.
3
Compare the key with elements in the sorted partition from right to left (j).
4
If the compared element is greater than the key, shift it one position to the right.
5
Once a smaller element is found (or the start is reached), insert the key into the correct spot. Repeat until done.
