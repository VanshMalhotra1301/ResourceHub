Binary Search

Best: O(1) Avg / Worst: O(log n) Space: O(1) Divide & Conquer

Binary Search efficiently locates a target in a sorted array by repeatedly halving the search space. Each comparison eliminates half the remaining elements — achieving logarithmic time where linear search would be O(n).

Pseudocode
function binarySearch(arr, target):
lo ← 0, hi ← len(arr) − 1
while lo ≤ hi:
mid ← ⌊(lo + hi) / 2⌋
if arr[mid] == target:
return mid ← FOUND ✅
else if arr[mid] < target:
lo ← mid + 1 ← search right →
else:
hi ← mid − 1 ← search left ←
return −1 ← NOT FOUND ❌

How Binary Search Works
1
Precondition: Array must be sorted in ascending order.
2
Set lo = 0 and hi = n−1. These mark the active search window.
3
Compute mid = ⌊(lo+hi)/2⌋. Compare arr[mid] to the target.
4
If arr[mid] == target → FOUND. If less → move lo right. If greater → move hi left.
5
Repeat until lo > hi. If loop exits without a match → NOT FOUND.
