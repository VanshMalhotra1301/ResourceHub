Kruskal's Algorithm (MST)

Time: O(E log E) Space: O(V + E) Greedy Approach

Kruskal's algorithm builds the Minimum Spanning Tree (MST) by sorting all edges by weight, then greedily picking the smallest edge. It uses a Disjoint Set (Union-Find) data structure to efficiently check if adding the edge would form a cycle.

Kruskal's Logic
function KruskalMST(Vertices, Edges):
mst ← []
uf ← new UnionFind(Vertices)
// Step 1: Sort all edges by weight
Edges.sort(ascending)
// Step 2: Iterate through sorted edges
for each (u, v, weight) in Edges:
// Step 3: Check for cycle using Find
root_u ← uf.find(u)
root_v ← uf.find(v)
if root_u != root_v: ← NO CYCLE
mst.append((u, v, weight))
uf.union(u, v)
else:
// Forms a cycle, discard edge
return mst

// C++ Kruskal's using Disjoint Set #include <iostream> #include <vector> #include <algorithm> using namespace std; struct Edge { int u, v, weight; bool operator<(Edge const& other) const { return weight < other.weight; } }; class UnionFind { private: vector<int> parent; public: UnionFind(int n) { parent.resize(n); for (int i = 0; i < n; i++) parent[i] = i; } int find(int i) { if (parent[i] == i) return i; return parent[i] = find(parent[i]); // Path compression } bool unite(int i, int j) { int root_i = find(i), root_j = find(j); if (root_i != root_j) { parent[root_i] = root_j; return true; } return false; } }; void kruskalMST(int V, vector<Edge>& edges) { UnionFind uf(V); vector<Edge> mst; sort(edges.begin(), edges.end()); for (Edge e : edges) { if (uf.unite(e.u, e.v)) { mst.push_back(e); } } }
