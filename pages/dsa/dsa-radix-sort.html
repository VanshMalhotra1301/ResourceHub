Radix Sort

Time: O(d * (n + b)) Space: O(n + b) Non-Comparative Sort

Radix Sort is an ingenious non-comparative algorithm that sorts elements digit by digit, from least significant to most significant. It uses a stable sub-sort (like Counting Sort, visualized here as 10 Buckets) for each digit pass.

Bucket Implementation Strategy
function radixSort(arr):
max_val ← getMax(arr)
for exp = 1 to max_val by exp*10:
buckets ← array of 10 empty lists
for each num in arr: ← DISTRIBUTE
digit ← ⌊(num / exp)⌋ % 10
buckets[digit].append(num)
idx ← 0
for each bucket in buckets: ← GATHER
for each num in bucket:
arr[idx++] ← num
return arr
Prefix-Sum Counting Approach
// Professional Radix Sort using Counting Sort void countingSort(vector<int>& arr, int exp) { int n = arr.size(); vector<int> output(n); vector<int> count(10, 0); // Count occurrences (Conceptual buckets) for (int i = 0; i < n; i++) count[(arr[i] / exp) % 10]++; // Prefix sums for stable positioning for (int i = 1; i < 10; i++) count[i] += count[i - 1]; // Build output array (Right to left for stability) for (int i = n - 1; i >= 0; i--) { output[count[(arr[i] / exp) % 10] - 1] = arr[i]; count[(arr[i] / exp) % 10]--; } for (int i = 0; i < n; i++) arr[i] = output[i]; } void radixSort(vector<int>& arr) { int m = *max_element(arr.begin(), arr.end()); for (int exp = 1; m / exp > 0; exp *= 10) countingSort(arr, exp); }
# Pythonic Radix Sort using literal Buckets (Visualizer Method) def radix_sort(arr): if not arr: return arr max_val = max(arr) exp = 1 while max_val // exp > 0: buckets = [[] for _ in range(10)] # Distribute into buckets based on current digit for num in arr: digit = (num // exp) % 10 buckets[digit].append(num) # Rebuild array from buckets arr.clear() for bucket in buckets: arr.extend(bucket) exp *= 10 return arr
// Professional Radix Sort using Counting Sort void countingSort(int arr[], int n, int exp) { int output[] = new int[n]; int count[] = new int[10]; Arrays.fill(count, 0); for (int i = 0; i < n; i++) count[(arr[i] / exp) % 10]++; for (int i = 1; i < 10; i++) count[i] += count[i - 1]; for (int i = n - 1; i >= 0; i--) { output[count[(arr[i] / exp) % 10] - 1] = arr[i]; count[(arr[i] / exp) % 10]--; } for (int i = 0; i < n; i++) arr[i] = output[i]; } void radixSort(int arr[], int n) { int m = Arrays.stream(arr).max().getAsInt(); for (int exp = 1; m / exp > 0; exp *= 10) countingSort(arr, n, exp); }
