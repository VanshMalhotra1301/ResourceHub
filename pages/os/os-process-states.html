State Explanations (Exam Review)

NEW

The process is being created. The OS sets up the PCB (Process Control Block) but hasn't moved the program into main memory yet.

READY

The process is fully in main memory and waiting to be assigned to a processor. It is queued in the Ready Queue.

RUNNING

Instructions are being executed by the CPU. In a single-core system, only ONE process can be in this state at any time.

BLOCKED / WAITING

The process cannot execute until some event occurs (e.g., I/O completion, waiting for disk/network). It relinquishes the CPU.

TERMINATED

The process has finished execution. The OS reclaims its memory and deletes its PCB.
