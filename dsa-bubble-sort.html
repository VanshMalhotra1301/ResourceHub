Bubble Sort

Best: O(n) Avg / Worst: O(n²) Space: O(1) Comparison Sort · Stable

Bubble Sort repeatedly steps through the array, compares adjacent pairs, and swaps them if they're in the wrong order. After each full pass, the largest unsorted element "bubbles up" to its correct position. The optimized version stops early if no swaps occur.

Pseudocode (Optimized)
function bubbleSort(arr):
n ← len(arr)
for i from 0 to n−2: ← pass i
swapped ← false
for j from 0 to n−i−2: ← compare pair
if arr[j] > arr[j+1]:
swap(arr[j], arr[j+1]) ← SWAP
swapped ← true
if not swapped: ← early exit
break ← already sorted!
Implementation
// Bubble Sort — O(n²) worst, O(n) best (optimized) void bubbleSort(vector<int>& arr) { int n = arr.size(); for (int i = 0; i < n - 1; i++) { bool swapped = false; for (int j = 0; j < n - i - 1; j++) { if (arr[j] > arr[j + 1]) { swap(arr[j], arr[j + 1]); swapped = true; } } if (!swapped) break; // already sorted } }

// Bubble Sort — O(n²) worst, O(n) best (optimized) void bubbleSort(int[] arr) { int n = arr.length; for (int i = 0; i < n - 1; i++) { boolean swapped = false; for (int j = 0; j < n - i - 1; j++) { if (arr[j] > arr[j + 1]) { int tmp = arr[j]; arr[j] = arr[j + 1]; arr[j + 1] = tmp; swapped = true; } } if (!swapped) break; } }

How Bubble Sort Works
1
Outer pass (i): Each pass guarantees the i-th largest element reaches its correct final position.
2
Inner compare (j): Compare arr[j] and arr[j+1]. If left > right → swap.
3
Bubble effect: Large values "bubble" rightward each pass. The sorted region grows from the right end.
4
Optimization: Track a swapped flag. If a full pass has 0 swaps, the array is sorted → break early.
5
Termination: After n−1 passes (or early exit), every element is in its correct place.
