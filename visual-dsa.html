Foundations & Sorting

Fundamental search and sort algorithms.

Binary Search

Efficient search using divide and conquer on a sorted array.

O(log N) O(1)

Bubble Sort

Repeatedly stepping through lists and swapping adjacent elements.

O(N²) O(1)

Selection Sort

Finding the minimum element and placing it at the beginning.

O(N²) O(1)

Insertion Sort

Building the final sorted array one item at a time.

O(N²) O(1)

Merge Sort

Divide and conquer algorithm that splits arrays and merges them.

O(N log N) O(N)

Quick Sort

Partitioning arrays using a pivot and sorting subarrays.

O(N log N) O(log N)

Radix Sort

Non-comparative integer sorting by processing individual digits.

O(d(N+K)) O(N+K)

Bucket Sort

Distributing elements into a number of buckets to sort individually.

O(N+K) O(N+K)

Greedy & Compression

Optimization processes making local optimal choices.

Fractional Knapsack

Maximizing total value dynamically using ratio sorting.

O(N log N) O(1)

Huffman Coding

Data compression using priority queues and optimal prefix trees.

O(N log N) O(N)

Activity Selection

Greedy approach to select non-overlapping activities.

O(N log N) O(1)

Kruskal's MST

Finding the minimum spanning tree using Disjoint Sets.
