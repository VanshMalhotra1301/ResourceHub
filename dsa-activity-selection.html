Activity Selection

Time: O(N log N) Space: O(1) Greedy Algorithm

Select the maximum number of non-overlapping activities. The optimal greedy strategy is to always pick the next available activity that finishes the earliest, thereby leaving the maximum possible time for subsequent activities.

# Python Greedy Activity Selection def select_activities(starts, finishes): n = len(starts) activities = [] for i in range(n): activities.append({'id': i, 's': starts[i], 'f': finishes[i]}) # Sort by finish time activities.sort(key=lambda x: x['f']) selected = [activities[0]['id']] last_finish = activities[0]['f'] for i in range(1, n): if activities[i]['s'] >= last_finish: selected.append(activities[i]['id']) last_finish = activities[i]['f'] return selected
