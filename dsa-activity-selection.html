Activity Selection Problem

Time: O(N log N) Space: O(1) Greedy Algorithm

A classic Greedy problem where you must select the maximum number of non-overlapping activities. The optimal strategy is to always pick the activity that finishes earliest.

Algorithm Logic
function selectActivities(S, F):
// Step 1: Sort by finish times
activities.sort(key=FinishTime)
selected = [activities[0]]
last_finish = activities[0].finish
for i from 1 to N-1:
if activities[i].start ≥ last_finish:
selected.add(activities[i])
last_finish = activities[i].finish
return selected
